# 階段 3 發現：橫斷面分析與方法論修正

**研究日期**：2026-05-19
**樣本**：71 個調入股票事件配對（v1 listwise drop 後 N=69；v2 N=62）
**版本說明**：本文檔同時保留 v1（已棄用）與 v2（採用）兩版結果，以記錄方法論修正過程
**主要產出**：`output/tables/cross_section_regression_v2.txt`、`output/figures/group_car_*_v2.png`

## 摘要

橫斷面分析揭示兩個關鍵發現：

1. **公告前 run-up 強度（CAR_pre_30）無法用事前可觀察特徵預測**：在 6 變數 OLS（N=62）中，所有變數 p > 0.25，R² = 0.072、adj-R² 為負。沒有任何規模、流動性、殖利率、動能、成交量異常等公開可得指標能顯著區分「哪些調入股的公告前 run-up 強」。
2. **公告後 reversal 強度（CAR_post_30）存在顯著橫斷面差異**：同樣 6 變數模型的 R² = 0.312、adj-R² = 0.237，三個變數顯著（殖利率 −、絕對成交量 −、相對周轉率 +）。

研究過程包含**兩次重大方法論修正**：

- **修正 (a)**：動能變數時間重疊 — 第一版用 `momentum_60d`（覆蓋公告前 60 天）與 `momentum_recent_5d`（公告前 5 天）預測 CAR_pre_30 [−30, 0]，視窗重疊 24 天，產生機械式自相關使 R² 假性膨脹至 0.345。
- **修正 (b)**：FinMind close=0 corporate action artifact 清理（見 `findings_stage2.md` §5.8）。意外揭示了 CAR_post_30 中被 outlier 遮蓋的真實訊號 — R² 從 0.214 上升至 0.312、多出一個顯著變數。

第一次修正讓「假預測」消失，第二次修正讓「真預測」浮現。兩者合起來提供階段 4 的設計依據：主軸仍應是「預測會被納入」（CAR_pre_30 無法差異化）；reversal 預測（CAR_post_30 模型）僅可作為次要章節。

## 2. 方法論修正

### 2.1 動能變數時間重疊修正

新動能變數定義：

| 新變數 | 計算視窗 | 與 CAR_pre_30 [−30, 0] 視窗關係 |
|---|---|---|
| `momentum_pre` | [−90, −31]（60 個交易日） | **結束於 day −31，完全位於 CAR 視窗之前** |
| `volume_anomaly_pre_5` | num = mean(vol[−30, −25])，den = mean(vol[−90, −31]) | volume 不是 return；雖然分子時間範圍位於 CAR 視窗內，但成交量與報酬是不同物理量，不會造成 return 自相關 |

`momentum_pre` 是嚴格「事前」的動能，可作為 CAR_pre_30 的正當預測變數。`volume_anomaly_pre_5` 捕捉「臨近公告的散戶提前布局留下的微結構痕跡」— 雖時間窗部分重疊，但測量的是量能而非報酬，方法論上可容許。

### 2.2 corporate action artifact 清理

詳見 `findings_stage2.md` §5.8。對橫斷面分析的具體影響：5347 世界先進在 event `00919_20240531` 的 CAR_post_30 在 v1 為假象 **−99.9 %**，在 v2 為真實 **+16.9 %**（清理後）。該觀測值在 v1 Cook's distance = 0.445（樣本最大值），對所有 CAR_post_30 迴歸係數構成系統性拖累。

### 4.1 為什麼清理 outlier 後 R² 反而上升 — Masked signal 現象

教科書中清理 outlier 通常使 R² 下降（outlier 提供大量變異），但本研究 v2 R² 從 0.214 上升到 0.312（+46 %）。這意味著 **5347 的假象 −99.9 % 不是「製造顯著性的雜訊」，而是「遮蓋真實訊號的雜訊」**。

Cook's distance Top-5 對照：

| 排名 | v1 (含 artifact) | v2 (清理後) |
|:---:|---|---|
| 1 | **5347 (00919_20240531) = 0.445** | 8422 (00919_20251216) = 0.374 |
| 2 | 5508 (00919_20240531) = 0.201 | 2348 (00919_20241217) = 0.106 |
| 3 | 1442 (00919_20241217) = 0.068 | 8112 (00919_20240531) = 0.070 |
| 4 | 8422 (00919_20251216) = 0.062 | 5508 (00919_20240531) = 0.059 |
| 5 | 2303 (00919_20221216) = 0.045 | 2211 (00919_20251216) = 0.053 |

5347 在 v1 是壓倒性 Top-1（Cook's d = 0.445，比第二名高 2.2 倍）。v2 中**完全掉出 Top-5**。v2 Top-5 觀測值的 Cook's d 都在合理範圍（最大 0.374），沒有單點極端影響。

## 5. 解讀

### 5.1 為什麼 CAR_pre_30 無法用事前特徵預測？

兩個非互斥的解釋：

**(a) 訊號太弱、N=62 樣本不足以檢出**

六個自變數對 62 個觀測值（10 obs / 1 var），對 OLS 來說自由度緊張。若真實 R² 在 0.10–0.20 範圍但個別 β 的標準誤大，所有 p 都可能落在 0.2–0.5 區間。樣本擴展（階段 4.3 的多 ETF 整合）能直接檢驗。

**(b) 事前布局基於非結構性訊號**

規則化選股的機制使「會被納入」的候選名單在公告前 1–2 週可被任何有方法論的市場參與者推算（見 `findings_stage2.md` §2.1）。但「哪些候選股會被推上去最多」可能不是由公開指標決定，而是由：

- 散戶情緒輪動（哪些股票最近被媒體 / 社群討論）
- 法人事前獲利了結時機（無從公開觀察）
- 個股特殊事件（季報、業績、產品消息）

這些訊號本質上不是橫斷面可觀察的，可能無法用規則化模型捕捉。

### 5.2 為什麼 CAR_post_30 反而可以差異化？

三個顯著變數，對應三個合理的**機制假說**（非已證實）：

| 變數 | 係數 | 機制假說 |
|---|---|---|
| `dividend_yield` | **負相關（β = −0.022）** | 高殖利率股代表「篩選時恰處於價格低點」，被納入後均值回歸力道更強 → 反向更深 |
| `log_avg_turnover` | **負相關（β = −0.084）** | 高絕對成交金額代表大型股 / 機構主力場域，機構在納入後獲利了結的時機協調性高 → 反向更深 |
| `turnover_ratio_60d` | **正相關（β = +6.62）** | 高相對周轉率代表散戶活躍 / 持股分散 → 沒有單一機構主導獲利了結 → 反向**較緩** |

機制相互一致：「**機構主導 + 大型股 + 高殖利率**」是反向最深的組合；「**散戶活躍 + 小型股**」反向最緩。

### 5.3 但要小心多重共線性

`log_avg_turnover`（成交金額 = 量 × 價）與 `turnover_ratio_60d`（相對周轉率 = 量 / 流通股數）在台股市場常呈現相反方向的相關：大型股絕對成交額高、但因股本大導致相對周轉率低。`dividend_yield` 與市值（雖未顯著但 β > 0）也部分相關 — 高息股通常是成熟、大型公司。

**這意味著三個顯著變數不應被視為三個獨立訊號**，而是同一個「機構持有大型成熟股 vs 散戶活躍成長股」維度的多面向投影。v2 報告未做正式 VIF 檢定，這是限制（見 §7.2）。

## 6. 對策略設計的影響

### 6.1 主軸不變

階段 2 確立的核心策略邏輯不變：

- **可獲利方向**：公告前進場、公告附近出場
- **前提條件**：能在公告前 5–10 天預測「會被新納入」
- **此預測能力是階段 4 的核心目標**

CAR_pre_30 的無預測力結論對此沒有負面影響 — 階段 4 預測的是「會不會被納入」（二元分類），不是「run-up 強度」（連續迴歸）。前者依靠對規則化選股方法論的理解 + 公開財務指標，是可行的；後者在本樣本失敗。

### 6.2 附帶發現：reversal 預測力的策略意涵

CAR_post_30 模型理論上可作為**反向放空策略訊號**：對「機構主導 + 大型股 + 高殖利率」的調入股，在生效日附近放空 30 天，預期 CAR 約 −10 % 至 −12 %（樣本中底層分位）。但實務障礙：

| 障礙 | 影響評估 |
|---|---|
| **借券成本**（年化 1–3 %） | 30 天約 0.1–0.3 % 成本 — 可接受 |
| **借券可得性** | 大型股有借券、小型股可能借不到。但訊號方向是「大型股反向更深」，與借券可得性相反方向疊加 → 中型股是甜蜜點 |
| **借券額度限制** | 同一檔個股借券池有限，多家機構同時放空會推升借券利率、稀釋 alpha |
| **散戶 / 個人投資人完全不可行** | 借券門檻、保證金、稅務處理 |

**評估**：這條路在新人 / 個人投資人層級**完全不可行**。對機構交易者（如優式資本）**理論上可行**，但需配合借券基礎建設與額度管理；且樣本 N=62、多重共線性未處理、未做 OOS 驗證，現階段不足以支持任何實際資金部署。

**結論**：列為階段 4 的「擴展研究」（階段 4.5 低優先級），非主軸。

## 7. 限制

### 7.1 樣本量 N = 62

多元迴歸自由度緊張（每變數約 10 觀測）。CAR_post_30 模型的三個顯著變數需要 N ≥ 100–150 才能確認穩健性。階段 4.3（多 ETF 樣本擴展）是直接驗證手段。

### 7.2 多重共線性未正式檢定

未報告 VIF、條件數、或變數間相關矩陣。三個顯著變數的經濟意義高度重疊（同一「機構 vs 散戶」維度的不同代理）。正式做法需以 PCA 或 ridge / lasso 重做迴歸；本研究未涵蓋。

### 7.3 單一 ETF

同 `findings_stage2.md` §5.2。本樣本完全來自 00919 高息 ETF 規則，外部效度未驗證。

### 7.4 未做穩健性檢定

未涵蓋：

- Robust standard errors（HC0 / HC3）
- Winsorized 版本（5 % / 95 %、1 % / 99 % 切位）
- Leave-one-out（剔除特定事件再重跑）
- Bootstrap CI for coefficients

此節為未來工作，不影響當前 v2 結論的方向，但會影響強度估計的可信區間。

### 7.5 產業效應未做正式檢定

71 觀測值散佈在 19 個產業，N ≥ 5 的產業僅 5 個（電子工業、金融保險、航運業、建材營造、半導體業，合計 41 觀測，占 57.7 %），其餘 14 個產業 N < 5。樣本過小使產業 fixed effect 迴歸不可行；產業效應僅以**探索性觀察**處理，不做正式統計推論。

## 8. 結論

橫斷面分析提供兩個對階段 4 有用的訊號：

1. **負面結論（穩健）**：CAR_pre_30 無法用事前可觀察特徵差異化。意味著「選股能力」不應作為策略的核心 alpha 來源；階段 4 應聚焦在二元分類（「會被納入」）而非連續迴歸（「run-up 強度」）。
2. **正面結論（待驗證）**：CAR_post_30 模型存在三個顯著變數（殖利率、絕對成交量、相對周轉率），R² = 0.312。但 N=62、多重共線性未處理、實務障礙明確，視為「橫斷面差異化的存在性證據」，而非可立即部署的預測模型。

**研究紀律的展示**：本階段經歷兩次重大方法論修正。

- 第一次（時間重疊）：讓 CAR_pre_30 的「假預測」（R² = 0.345）消失。
- 第二次（corporate action artifact）：讓 CAR_post_30 的「真預測」（R² 從 0.214 上升到 0.312）浮現。

這個雙向影響展示了量化研究中「資料品質」與「變數定義」的對等重要性：前者解決「值是否真實」，後者解決「測量是否有效」。任一環節的問題都會雙向扭曲結論 — **第一次是把不存在的訊號做成存在；第二次是把存在的訊號被遮蓋**。
